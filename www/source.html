Note

The source codes for RemotePad and RemotePad Servers are available with following methods. Almost of these codes are BSD-style open source, so you can see the codes, compile applications by yourself, or modify applications. Any suggestions or code improvements are welcome all the time. Moreover you can also use the codes for any porposes. If you have any questions about using our codes, please contact us at RemotePad-request@tenjin.org.

Download Source Packages

  • Please download from Downloads page in Google Code.

Browse the Source Code

  • Browse the source code repository provided by Google Code.
  • View the source code changes history provided by Google Code.

Command-Line Access

  • You can anonymously check out the latest project source code like this: 
    	svn checkout http://iphone-remotepad.googlecode.com/svn/trunk/ iphone-remotepad-read-only

RemotePad Server for Windows and for X11 are based on iTouchpad sources. Thanks Will Dietz!
